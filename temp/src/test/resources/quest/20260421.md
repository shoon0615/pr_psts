## 문법

예제 및 과제는 없습니다.

### 메소드 체이닝

`기존`

```java
for (Member member : list) {
    if (member.getId() == 2) {
        return member.getName();
    }
}
return null;
```

`변경`

```java
return list.stream()
        .filter(member -> member.getId() == 2)
        .map(Member::getName)
        .findFirst()
        .orElse(null);
```

### 메소드 참조(축약)

`기존`

```java
Stream.of().method((args) => System.out.println(args))
```

`변경`

```java
Stream.of().method(System.out::println)
```

## interface

예제 및 과제는 없습니다.  
`기존` @Override 로만 처리 가능

```java
class Class implements Interface {
	@Override
	boolean method() { ... }
}
```

`변경` @Override 외에도 기능의 다양화

```java
interface Interface {
   default boolean method1() { ... }
   private boolean method2() { ... }
   static boolean method3() { ... }
}
```

`추가` 접근 제한 기능

```java
sealed interface Interface permits Class1, Class2 { ... }
sealed class Interface permits Class1, Class2 { ... }
```

## Switch

예제 및 과제는 없습니다.  
`기존`

```java
switch (param) {
	case PARAM1:
		return result;
		break;
	case PARAM2:
	case PARAM3:
		break;
	default:
		break;
}
```

`java16` 화살표 문법 추가

```java
switch (param) {
    case PARAM1 -> result;
    case PARAM2, PARAM3 -> { yield result; };
    default -> { yield result; }
}
```

`java21` instanceof 및 조건문 등 다양한 패턴 추가

```java
switch (param) {
    case null -> result;
    case String s when s.length() > 10 -> { yield result; };
    case String s when s.length() < 3 -> { yield result; };
    default -> { yield result; }
}
```
